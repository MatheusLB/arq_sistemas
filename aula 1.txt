web services solucoes que se comunicam independente de linguagem, software e hardware

utilizam linguagem de marcação, inicialemte comecou com XML

permite integracao entre aplicações via troca de mensagens (xml, jason) entre o servidor ou web ser

Expoe uma uri para que a aplicacao consiga acesso e liberar os dados para o webservice

Toda comunicacao e feita pelo protocolo http: aplicaccao y manda uma requisicao para o web service, onde o mesmo ira interpretar, acessar o banco de dados da aplicacao x
e devolver para o y...rapide na integracao

vantagens: linguagem comum, facilita integracao, melhora segurança vc trata o que vai devolver de dados ou não
