internet das coisas

embutir sensores em objetos do dia a dia

coletar dados dos sensores

analisar dados

3 fases:
things: coisas onde sao colocados sensores

cloud: nuvem para processar os dados

intelligence: utlizar para otimizar processaos, vantagem competitiva

smart building: capacidade de coletar dados de componente para serem usados de forma inteligencia

smart home: diversos sensores pela casa como por exemplo, sensor de temperatura que ao verificar o portao abrindo, por ligar o ar condicionado 

wearables: componente vestiveis: relógio

agriculture: sensores que medem a humidade do solo

smart transportation: sensores para comunicação entre veiculos

computação ubíqua: dispositivos conectados a rede se comunicando entre si, distansiando de usuarios casuiasi

Desafios: 
1privacidade e segurança
2quantidade exponencial de dispositivos
3capacidade de processar e armazenar grande quantidade de informações
4gerar valor a partir dos dados
